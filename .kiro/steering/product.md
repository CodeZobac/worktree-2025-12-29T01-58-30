# Product Overview

Family Recipes is a web application for sharing and discovering family recipes. Users can create, organize, and browse recipes within their family group.

## Core Features

- User authentication via Google OAuth
- Family group management
- Recipe creation and organization with folders
- Markdown support for recipe instructions
- Image upload and optimization for recipe photos
- 3D recipe card displays with animations

## User Flow

Users must authenticate to access the app. After login, they can join or create a family group, then create and organize recipes within folders. Recipes support rich markdown formatting and image attachments.
